## 사용 가능한 MCP 도구들

### 🧠 AI Brain
- `compute_thought` - 하이브리드 추론 (BitNet + Cloud)
- `compute_recursive_thought` - 심층 재귀 추론
- `get_brain_status` - 브레인 상태 확인

### 🎵 Music Production
- `create_full_song` - 풀 파이프라인 음악 생성
- `generate_music_lyrics` - 가사 생성
- `check_engine_status` - 음악 엔진 상태

### 🎨 Visual Creation
- `route_visual_task` - 이미지/비디오 자동 라우팅
- `generate_image` - Flux.1-Schnell 이미지 생성
- `generate_video` - CogVideoX 비디오 생성
- `analyze_image_file` - 이미지 분석

### 📁 Filesystem
- 전체 드라이브 (C, D, E, F, G, I) 읽기/쓰기 가능

### 🐙 GitHub
- 레포지토리 검색, 코드 분석, PR 관리

### 💾 Supabase
- 데이터베이스 쿼리, 마이그레이션

### 🔍 Everything
- 초고속 파일 검색

### 🧬 Engram
- 시맨틱 메모리 저장/검색

### 📚 Context7
- 라이브러리 문서 검색
