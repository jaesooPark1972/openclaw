# Telegram 듀얼 봇 운영 규칙

## 1) 그룹 구성
- 주인님 + 현재 봇(OpenClaw) + 학원 봇 초대
- 개인 대화 대신 그룹에서 공동 운영

## 2) 명령어 분리
- 현재 봇: `/kkoma ...`
- 학원 봇: `/academy ...`

## 3) 역할 분담
- 현재 봇: 총괄 요약/보고/리포트
- 학원 봇: 학원 전용 실행/현장 작업

## 4) 안전 규칙
- 위험 명령(송금/삭제/재시작)은 확인 후 실행
- API 키/토큰은 그룹에 업로드 금지

## 5) 시작 테스트
1. `/kkoma 상태`
2. `/academy 상태`
3. `/kkoma 오늘요약`
4. `/academy 실행로그`
5. 응답 정상 시 운영 시작
