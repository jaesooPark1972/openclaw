# ðŸ“§ Gmail Control Skill

This skill allows the agent to interact with the user's Gmail account directly.

## Capabilities

1.  **Read Emails**: Can list unread emails or search for specific messages (`list_emails`), and read the full content of any email (`read_email`).
2.  **Send Emails**: Can send emails to any recipient (`send_email`).
3.  **Create Drafts**: Can create draft emails for later review or sending (`create_draft`).

## Setup Requirements

To use this skill, the system must be authenticated with Google OAuth 2.0.

1.  **Credentials**: `f:\AGen\keys\gmail_credentials.json` (OAuth Client ID) must exist.
2.  **Token**: `f:\AGen\keys\gmail_token.json` is generated by running the setup script.
3.  **Setup Script**: Rune `python f:\AGen\scripts\setup_gmail.py` to authenticate.

## Example Usage

### specific: Check unread emails from "Boss"
```python
emails = list_emails(query="from:boss is:unread")
print(emails)
if "Meeting" in emails:
    read_email(message_id="...")
```

### specific: Draft a response
```python
create_draft(to="client@example.com", subject="Project Update", body="Here is the latest report...")
```
